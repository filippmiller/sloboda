# ДОМЕН 20: УПРАВЛЕНИЕ РИСКАМИ (RSK)

**Проект**: СЛОБОДА — автономное поселение
**Версия**: 1.0
**Дата**: 2026-02-13
**Масштаб**: 300 домов (базовый), масштабируемо до 500

---

## МЕТОДОЛОГИЯ

Документ создан на основе брейншторма 5 специалистов (30 итераций):
1. **Специалист по управлению рисками** — идентификация рисков, матрица вероятность/ущерб, стратегии снижения
2. **Страховой специалист** — страхуемые vs нестрахуемые риски, резервные фонды, программы страхования
3. **Эксперт по кризисному управлению** — протоколы реагирования, эвакуация, восстановление
4. **Консультант по безопасности** — физическая, кибер-, информационная безопасность
5. **Планировщик непрерывности деятельности** — BCP, резервирование, цепочки поставок

## 20.1 ПРИРОДНЫЕ РИСКИ

### R-NAT-01: Засуха

| Параметр | Значение |
|----------|----------|
| Вероятность | 3/5 (раз в 3–5 лет в средней полосе) |
| Ущерб | 4/5 (потеря 30–60% урожая открытого грунта) |
| Балл риска | 12 |

**Митигация:**
- Артезианские скважины с запасом дебита 30% (не зависят от осадков)
- Каскад прудов для ирригации (2 500 м³ — домен VOD)
- Капельный полив на всех посадках (экономия воды 40–60%)
- Мульчирование (снижение испарения на 50%)
- Засухоустойчивые сорта в основном севообороте
- Теплицы с замкнутым водооборотом — не зависят от погоды
- **Стратегический запас**: продовольствие на 3 мес (см. R-RES ниже)

### R-NAT-02: Суровая зима (ниже -35°C, более 2 недель)

| Параметр | Значение |
|----------|----------|
| Вероятность | 2/5 |
| Ущерб | 4/5 (замерзание труб, гибель животных, перерасход топлива) |
| Балл риска | 8 |

**Митигация:**
- Дома рассчитаны на -30°C с запасом (стены R=5.0 — домен DOM)
- Запас топлива на 150% нормы (домен ENR)
- Водопроводные трубы на глубине 2.0–2.2 м (ниже промерзания)
- Электрический обогрев критических участков водопровода
- Утеплённые помещения для животных (домен ZHV)
- Резервные генераторы (потеря электричества при морозе критична)
- **Дежурная бригада**: круглосуточный мониторинг систем отопления зимой

### R-NAT-03: Паводок / Наводнение

| Параметр | Значение |
|----------|----------|
| Вероятность | 2/5 (зависит от выбора участка) |
| Ущерб | 5/5 (потеря зданий и инфраструктуры) |
| Балл риска | 10 |

**Митигация:**
- **Главная мера**: выбор участка на возвышенности (домен GEN, критерий выбора)
- Не строить в пойменных зонах
- Дренажная система и ливневая канализация (домен VOD)
- Каскадные пруды как буфер
- Если участок рядом с рекой — минимум 200 м от уреза воды

### R-NAT-04: Ураган / Шквал

| Параметр | Значение |
|----------|----------|
| Вероятность | 2/5 |
| Ущерб | 3/5 (повреждение кровель, теплиц, линий электропередач) |
| Балл риска | 6 |

**Митигация:**
- Ветрозащитные полосы (30–50 м, северная и западная стороны — домен GEN)
- Кровля с усиленным крепежом (шаг обрешётки, количество саморезов)
- Теплицы: арочная конструкция устойчивее двускатной
- Резервные листы поликарбоната / профлиста на складе
- **Протокол**: при штормовом предупреждении — закрытие теплиц, укрепление навесов

### R-NAT-05: Неурожай

| Параметр | Значение |
|----------|----------|
| Вероятность | 3/5 (по отдельным культурам — ежегодно) |
| Ущерб | 3/5 (дефицит отдельных продуктов) |
| Балл риска | 9 |

**Митигация:**
- Диверсификация: 15+ культур (если одна провалится — остальные компенсируют)
- Теплицы: контролируемая среда, минимальная зависимость от погоды
- 4–5 летний севооборот (восстановление почвы)
- Семенной фонд: собственное семеноводство + резервный запас покупных
- Стратегический запас овощей на 3 мес в овощехранилище
- **При тотальном неурожае**: закупка на рынке + помощь от сети поселений (домен SET)

### R-NAT-06: Лесной / Степной пожар

| Параметр | Значение |
|----------|----------|
| Вероятность | 2/5 |
| Ущерб | 5/5 |
| Балл риска | 10 |

**Митигация:**
- Минерализованная полоса 10–15 м по периметру поселения
- Противопожарные пруды (700 м³ — домен POZ)
- Добровольная пожарная дружина (20–25 чел — домен POZ)
- Мотопомпы, рукава, огнетушители
- Система раннего обнаружения (наблюдатели + радиосвязь)

## 20.2 ТЕХНОГЕННЫЕ РИСКИ

### R-TEC-01: Пожар в поселении

| Параметр | Значение |
|----------|----------|
| Вероятность | 4/5 (деревянные дома + печное отопление) |
| Ущерб | 4/5 (потеря дома / нескольких домов) |
| Балл риска | 16 — **КРИТИЧЕСКИЙ** |

**Митигация:**
- Противопожарные разрывы 18–20 м между домами (домен POZ)
- Обработка всех каркасов огнебиозащитой (Неомид 450-1)
- Дымовые извещатели в каждом доме (минимум 2)
- Правильная установка печей/котлов (отступы, разделки, отмостки)
- Противопожарная дружина — время прибытия 5 мин (домен POZ)
- Страхование каждого дома (см. раздел «Страховая стратегия»)
- **Регулярные проверки**: дымоходы — 2 раза/год, электропроводка — 1 раз/год

### R-TEC-02: Поломка ключевого оборудования

| Параметр | Значение |
|----------|----------|
| Вероятность | 4/5 (механика изнашивается) |
| Ущерб | 3/5 (простой производства) |
| Балл риска | 12 |

**Критическое оборудование (отказ которого останавливает процесс):**
1. Пилорама → нет доски → нет домов
2. Сушильная камера → нет сухой доски → нет комплектов
3. Скважинный насос → нет воды
4. Генератор → нет электричества
5. Котёл теплицы → гибель растений зимой

**Митигация:**
- **Резервирование**: 2 пилорамы (если 1 сломалась — работает вторая)
- **ЗИП**: запасные части на каждый станок (ремни, подшипники, пилы, ножи)
- **Контракт на сервис**: с поставщиком оборудования (выезд в течение 48 ч)
- **Обученный механик**: в штате поселения (домен UPR)
- **Профилактика**: ежемесячное ТО по графику (домен PRO)
- Для насосов: 2-й резервный насос на складе
- Для генераторов: 2 генератора (основной + резервный)

### R-TEC-03: Отключение электричества

| Параметр | Значение |
|----------|----------|
| Вероятность | 3/5 |
| Ущерб | 3/5 (остановка производства, холодильники, насосы) |
| Балл риска | 9 |

**Митигация:**
- 2 генератора (основной + резервный)
- Газогенераторная ТЭЦ как основной источник (домен ENR, Фаза 7)
- ИБП (UPS) на сервер и медоборудование
- Гравитационное водоснабжение (водонапорная ёмкость — работает без электричества)
- Печное/котловое отопление не зависит от электричества (кроме циркуляционного насоса — предусмотреть гравитационную схему)

### R-TEC-04: Авария водоснабжения

| Параметр | Значение |
|----------|----------|
| Вероятность | 2/5 |
| Ущерб | 4/5 (без воды — эвакуация за 3 дня) |
| Балл риска | 8 |

**Митигация:**
- 10 скважин (выход из строя 1–2 не критичен — домен VOD)
- Кольцевая схема водопровода (можно отключить повреждённый участок)
- Аварийный запас воды: 600 м³ в ёмкостях (3 дня на всё поселение)
- Каскадные пруды: техническая вода для полива и пожаротушения
- Аварийная бригада: 2 человека с навыком ремонта ПЭ-труб
- Запас труб и фитингов на складе

### R-TEC-05: Авария канализации

| Параметр | Значение |
|----------|----------|
| Вероятность | 2/5 |
| Ущерб | 3/5 (загрязнение, санитарная угроза) |
| Балл риска | 6 |

**Митигация:**
- Кластерные ЛОС (3 независимых — выход одной не критичен)
- Регулярная откачка и обслуживание (график — домен VOD)
- Запас хлорной извести для аварийной дезинфекции
- Биотуалеты (10 шт на складе — временная мера при аварии)

|----------|----------|

| Балл риска | 15 — **КРИТИЧЕСКИЙ** |

**Митигация:**
- 5-ступенчатая система разрешения конфликтов (домен UPR)

- Регулярные общие собрания (еженедельно) — площадка для открытого обсуждения
- Анонимная система обратной связи
- Чёткие правила внутреннего распорядка (утверждены Сходом)
- Процедура исключения из общины как крайняя мера (2/3 голосов)
- **Профилактика**: совместные праздники, ротация бригад, общие проекты

### R-SOC-02: Отток населения

| Параметр | Значение |
|----------|----------|
| Вероятность | 4/5 (особенно в первые 2 года) |
| Ущерб | 4/5 (потеря специалистов, незанятые дома, нарушение экономики) |
| Балл риска | 16 — **КРИТИЧЕСКИЙ** |

**Митигация:**
- Тщательный отбор жителей (испытательный срок 3–6 мес — домен UPR)
- Привлекательные условия: дом за 2 млн руб (vs 5–8 млн за сопоставимый в городе)
- Развитая социальная инфраструктура (школа, медпункт, клуб — домен SOC)
- Комфортное жильё (не «времянки», а полноценные дома с удобствами)
- Возможность удалённой работы (интернет — домен ITS)
- Финансовая привязка: возвратный взнос с удержанием при раннем выходе
- **Индикатор раннего обнаружения**: ежеквартальный опрос удовлетворённости

### R-SOC-03: Нехватка ключевых специалистов

| Ущерб | 3/5 |

**Критические специальности:**
- Фельдшер / медсестра

- Механик / слесарь

- Агроном
- Ветеринар

**Митигация:**
- Программа «Земский учитель» / «Земский фельдшер» (1 млн руб подъёмных)
- Предоставление бесплатного жилья специалистам
- Многопрофильное обучение (каждый осваивает 2+ специальности — домен UPR)
- Договоры с приезжающими специалистами (1–2 раза в месяц)
- Телемедицина для консультаций (домен MED)
- **Обязательное наставничество**: каждый специалист готовит замену (домен UPR)

|----------|----------|

- Замещение: если цемент дорожает — больше деревянных конструкций

- **Минимальная жизнеспособная конфигурация**: 50 домов + завод + минимум с/х = ~150 млн руб

## ТОП-10 КРИТИЧЕСКИХ РИСКОВ: ДЕТАЛЬНЫЕ ПЛАНЫ

### 1. R-ECO-02: Недостаток финансирования (балл 20)
**Триггер**: остаток на счету < 3 мес операционных расходов
**План действий**:
1. Ускорить продажу домокомплектов (маркетинг, скидки)
2. Подать заявки на гранты (домен EKO — 7 программ)
3. Объявить дополнительный набор жителей (взносы)
4. Заморозить нескритическое строительство (социалка, расширение)
5. Обратиться к сети поселений за финансовой помощью
6. Рассмотреть кредитование под залог земли/оборудования
**Ответственный**: Финансовый комитет
**Мониторинг**: еженедельный отчёт по cash flow

### 2. R-TEC-01: Пожар в поселении (балл 16)
**Триггер**: любое возгорание
**План действий**:
1. Тревога (радиосвязь) — дежурная смена выезжает (3 мин)
2. Развёртывание мотопомпы, подключение к пруду (5 мин)
3. Эвакуация людей из зоны риска
4. Локализация: не дать огню перейти на соседние дома
5. Вызов районной пожарной части (при необходимости)
6. После тушения: осмотр, акт, начало восстановления
**Ответственный**: Начальник ДПД
**Мониторинг**: ежемесячные учения, ежегодная проверка дымоходов

### 3. R-SOC-02: Отток населения (балл 16)
**Триггер**: 5+ семей заявили о намерении уехать в квартал
**План действий**:
1. Индивидуальные беседы (выяснить причины)
2. Если причина системная — экстренное собрание Совета
3. Разработка плана устранения проблемы (в течение 2 недель)
4. Если дефицит кадров — срочный набор (через сеть поселений)
5. Пересмотр условий, если нужно
**Ответственный**: Комитет по социальным вопросам
**Мониторинг**: ежеквартальный опрос удовлетворённости (шкала 1–10)

### 4. R-SOC-01: Конфликты (балл 15)
**Триггер**: конфликт не разрешён на 2-й ступени (соседская медиация)
**План действий**: 5-ступенчатая эскалация (домен UPR)
**Ответственный**: Комитет медиации
**Мониторинг**: журнал конфликтов, ежеквартальный анализ

### 5. R-ECO-01: Рост цен на материалы (балл 15)
**Триггер**: рост цен > 15% за квартал
**План действий**:
1. Увеличить закупки впрок (до подорожания)
2. Пересмотреть BOM: чем можно заменить подорожавшее?
3. Увеличить долю собственного производства
4. Скорректировать бюджет фазы
**Ответственный**: Финансовый комитет + Снабженец
**Мониторинг**: ежемесячный мониторинг цен на ключевые материалы

### Итерация 30: Каскадные отказы, продовольственная безопасность и ежегодная ревизия рисков

> **Планировщик непрерывности деятельности**: «Самые страшные аварии — каскадные. Не "сломалась пилорама", а "сломалась пилорама → нет досок → не починили крышу теплицы → погиб урожай → дефицит продовольствия → паника → отток". Нужно анализировать не отдельные события, а цепочки.»

> **Специалист по управлению рисками**: «Финальная итерация посвящена трём вещам: каскадным отказам (что если несколько рисков реализуются одновременно), комплексной продовольственной безопасности и процессу, который обеспечит актуальность этого документа на протяжении десятилетий — ежегодной ревизии рисков.»

#### ЧАСТЬ А: Анализ каскадных отказов

**Методология:** для каждой критической системы определяем, какие другие системы зависят от неё (граф зависимостей), и рассчитываем совокупный ущерб от каскада.

**Граф критических зависимостей:**

```
ЭЛЕКТРИЧЕСТВО ─→ Водоснабжение (насосы)
      │           ├→ Теплицы (климат-контроль)
      │           ├→ Животноводство (поилки)
      │           └→ Продовольствие
      ├→ IT-системы (сервер, связь)
      │   └→ Управление (данные, координация)
      ├→ Производство (пилорама, станки)
      │   └→ Доходы → Финансы
      └→ Медицина (ФАП — оборудование, холодильник)

ВОДОСНАБЖЕНИЕ ─→ Жилые дома (санитария)
      │          ├→ Теплицы (полив)
      │          ├→ Животноводство
      │          └→ Пожаротушение (!!)
      └→ Если нет воды + нет пожаротушения = критическая уязвимость

ТОПЛИВО ─→ Транспорт (связь с городом)
    │       ├→ Снабжение (поставки)
    │       └→ Эвакуация (медицинская)
    ├→ Генераторы (резервное электричество)
    └→ Отопление (часть домов)
```

**Сценарии одновременной реализации рисков:**

| Сценарий | Комбинация | Совокупный ущерб | Вероятность |
|----------|-----------|-----------------|-------------|
| «Чёрная зима» | Суровая зима + отключение электричества + поломка генератора | Замерзание труб, гибель скота, эвакуация | 1/5 x 3/5 x 2/5 = **очень низкая, но ущерб катастрофический** |
| «Голодный год» | Засуха + эпизоотия + разрыв поставок | Дефицит продовольствия 60%+ | 2/5 (составляющие коррелированы) |
| «Идеальный шторм» | Пожар + авария водоснабжения | Нечем тушить → потеря нескольких домов | 1/5, но ущерб 5/5 |
| «Цифровой паралич» | Кибератака + отказ связи | Потеря управления + нет внешней помощи | 1/5 |
| «Социальный коллапс» | Финансовый кризис + массовый отток + конфликт | Распад общины | 2/5 (составляющие коррелированы) |

**Противокаскадные меры:**

| Мера | Описание | Какие каскады прерывает |
|------|----------|------------------------|
| Тройное резервирование электричества | ТЭЦ + дизель + мобильный генератор | Все каскады через электричество |
| Гравитационное водоснабжение | Водонапорная башня на 100 м³ — работает без электричества | Электричество → Вода |
| 3-месячный продовольственный запас | Автономность от внешних поставок и от урожая | Любые продовольственные каскады |
| Автономная радиосвязь (DMR) | Работает без интернета и электросети (аккумуляторы) | IT → Управление |
| Противопожарный пруд с самотёчным подключением | Вода для тушения без насосов | Вода → Пожаротушение |
| Финансовая подушка 6 мес OPEX | Время для решения любого кризиса | Финансовые каскады |

#### ЧАСТЬ Б: Комплексная продовольственная безопасность

**Сценарий «Тройной удар»: неурожай + падёж скота + разрыв поставок**

| Параметр | Значение |
|----------|----------|
| Вероятность (одновременно) | 1/5 |
| Ущерб | 5/5 (голод, эвакуация, потеря проекта) |
| Балл риска | **5** (низкий, но последствия неприемлемы) |

**Многоуровневая система продовольственной безопасности:**

| Уровень | Источник | Покрытие | Автономность |
|---------|---------|---------|--------------|
| 1. Текущее производство | Огороды, теплицы, животные | 70–80% потребности | Полная |
| 2. Стратегический запас | Зернохранилище, овощехранилище, консервы | 90 дней на 100% потребности | Полная |
| 3. Сеть поселений | Взаимовыручка, обмен продукцией (домен SET) | Дополнительно 30–60 дней | Частичная |
| 4. Внешний рынок | Закупки в городе, оптовые базы | Неограниченно при наличии денег | Зависимая |
| 5. Государственная помощь | МЧС, гуманитарная помощь | Неопределённо | Зависимая |

**Протокол «Продовольственная тревога»:**

| Уровень | Триггер | Действия |
|---------|---------|----------|
| **ЗЕЛЁНЫЙ** | Запас > 90 дней | Нормальный режим |
| **ЖЁЛТЫЙ** | Запас 60–90 дней или прогноз неурожая | Увеличить закупки, усилить теплицы |
| **ОРАНЖЕВЫЙ** | Запас 30–60 дней | Рационирование необязательных продуктов, срочные закупки |
| **КРАСНЫЙ** | Запас < 30 дней | Строгое рационирование, мобилизация всех источников |

**Рационирование (КРАСНЫЙ уровень) — норма на 1 человека в день:**

| Продукт | Норма | Калорийность |
|---------|-------|-------------|
| Хлеб / мука (в пересчёте) | 500 г | 1 200 ккал |
| Картофель / крупа | 300 г | 300 ккал |
| Масло растительное | 30 г | 270 ккал |
| Молоко / кисломолочные | 200 мл | 120 ккал |
| Мясо / рыба / бобовые | 100 г | 200 ккал |
| Овощи / фрукты | 200 г | 80 ккал |
| **Итого** | | **~2 170 ккал** |

Это минимум для выживания взрослого человека. Дети, беременные, кормящие — приоритет с увеличенными нормами.

#### ЧАСТЬ В: Процесс ежегодной ревизии рисков

**Цель:** обеспечить актуальность и полноту документа RSK на протяжении всей жизни поселения.

**Периодичность:** 1 раз в год, в январе (после окончания годового цикла).

**Ответственный:** Комитет по рискам (3–5 человек: председатель Совета + финансист + механик + агроном + медик).

**Процедура ежегодной ревизии:**

| Этап | Действие | Срок | Ответственный |
|------|----------|------|---------------|
| 1. Сбор данных | Инциденты за год, изменения среды, новые угрозы | 1–15 января | Секретарь |
| 2. Анализ инцидентов | Разбор каждого ЧП: причина, реакция, уроки | 15–20 января | Комитет |
| 3. Пересчёт матрицы | Обновить вероятности и ущербы на основе фактических данных | 20–25 января | Риск-менеджер |
| 4. Новые риски | Идентификация рисков, не учтённых ранее | 25–30 января | Комитет |
| 5. Проверка митигаций | Все меры на месте? Резервы пополнены? Оборудование работает? | 1–7 февраля | Бригадиры |
| 6. Проверка страхования | Полисы продлены? Покрытие достаточно? | 7–10 февраля | Бухгалтер |
| 7. Обновление документа | Внести изменения в RSK README | 10–15 февраля | Секретарь |
| 8. Утверждение | Общее собрание — доклад о рисках и утверждение плана | Февральский Сход | Председатель |

**Формат отчёта по ежегодной ревизии:**

```
ЕЖЕГОДНЫЙ ОТЧЁТ ПО РИСКАМ — [ГОД]

1. ИНЦИДЕНТЫ ЗА ГОД
   - [Дата] — [Описание] — [Ущерб] — [Реакция] — [Уроки]

2. ИЗМЕНЕНИЯ МАТРИЦЫ
   - R-XXX-XX: вероятность изменена с X на Y (причина: ...)
   - Новый риск: R-XXX-XX — описание

3. СОСТОЯНИЕ РЕЗЕРВОВ
   - Продовольственный: XX дней (норма: 90)
   - Топливный: XX дней (норма: 60)
   - Финансовый: XX мес OPEX (норма: 3)

4. СОСТОЯНИЕ СТРАХОВАНИЯ
   - Все полисы действительны: ДА / НЕТ
   - Непокрытые риски: ...

5. ПЛАН НА СЛЕДУЮЩИЙ ГОД
   - Приоритетные меры: ...
   - Бюджет на митигацию: ... руб

6. УТВЕРЖДЕНО
   Сход поселения, [дата], протокол № ...
```

**Ключевые метрики для ежегодного отслеживания:**

| Метрика | Единица | Целевое значение |
|---------|---------|-----------------|
| Количество ЧП за год | шт | < 5 |
| Среднее время реагирования | часы | < 2 |
| Запас продовольствия | дни | > 90 |
| Запас топлива | дни | > 60 |
| Финансовый резерв | мес OPEX | > 3 |
| Bus factor ключевых позиций | чел | >= 2 |
| Удовлетворённость жителей | балл (1–10) | > 7 |
| Действующие страховые полисы | % от плана | 100% |
| Проведённые учения | шт/год | >= 4 |
| Конфликты, разрешённые на 1–2 ступени | % | > 90% |

> **Страховой специалист**: «Ревизия рисков — это не бюрократия. Это момент, когда поселение останавливается и честно спрашивает себя: "Мы готовы к тому, что может произойти в следующем году?" Если ответ "нет" — значит, ревизия уже окупила себя.»

> **Планировщик непрерывности деятельности**: «Документ RSK — это живой организм. Если его не обновлять, через 3 года он станет бесполезной бумагой. Ежегодная ревизия — это не рекомендация, а обязательство перед общиной. Каждый житель имеет право знать, к каким рискам поселение готово, а к каким — ещё нет.»
