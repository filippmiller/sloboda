### 4.1.5 Сравнение вариантов отопления

#### Вариант 1: Индивидуальные пеллетные котлы

**Описание**: Каждый дом оснащён автоматическим пеллетным котлом с бункером. Водяная система отопления (радиаторы или тёплый пол).

| Параметр | Значение |
|----------|----------|
| Тип котла | Автоматический пеллетный, 10-20 кВт |
| КПД | 85-92% |
| Бункер | 200-500 л (2-7 дней автономной работы) |
| Расход пеллет (дом B, 80 м2) | ~4.5 тонн/сезон |
| Обслуживание | Чистка золы 1-2 раза/неделю, ТО раз в год |
| Стоимость котла | 150 000 - 350 000 руб |
| Система отопления (радиаторы, трубы, обвязка) | 100 000 - 200 000 руб |

**Плюсы**: Полная автономность каждого дома, нет теплотрассы, простое масштабирование.
**Минусы**: Требует хранения пеллет у каждого дома, необходимо обслуживание каждым жителем.

#### Вариант 2: Индивидуальные пиролизные котлы длительного горения

**Описание**: Котлы на дровах/брикетах с верхним горением или газогенерацией. Одна закладка горит 8-24 часа.

| Параметр | Значение |
|----------|----------|
| Тип котла | Пиролизный (газогенераторный), 15-25 кВт |
| КПД | 80-90% |
| Длительность горения на 1 закладке | 8-14 часов (дрова), до 24 ч (брикеты) |
| Расход дров (дом B, 80 м2) | ~12 м3/сезон (берёзовых, колотых) |
| Обслуживание | Закладка 1-3 раза/сутки, чистка еженедельно |
| Стоимость котла | 80 000 - 200 000 руб |
| Система отопления | 100 000 - 200 000 руб |

**Плюсы**: Работает на любой древесине, дешевле пеллетного, использует местное сырьё.
**Минусы**: Ручная загрузка, требует дровника у каждого дома, менее комфортно.

#### Вариант 3: Индивидуальные котлы на щепе

**Описание**: Автоматические котлы с подачей щепы из бункера.

| Параметр | Значение |
|----------|----------|
| Тип котла | Шнековая подача щепы, 15-30 кВт |
| КПД | 82-88% |
| Бункер | 3-5 м3 (5-10 дней автономной работы) |
| Расход щепы (дом B, 80 м2) | ~15 м3/сезон |
| Стоимость котла | 250 000 - 500 000 руб |

**Плюсы**: Автоматическая работа, использует отходы лесопилки.
**Минусы**: Дорогое оборудование, крупный бункер, влажность щепы критична.

#### Вариант 4: Кирпичная печь (печное отопление)

**Описание**: Традиционная русская печь или печь-камин с водяной рубашкой.

| Параметр | Значение |
|----------|----------|
| Тип | Кирпичная печь (русская, шведка, голландка) |
| КПД | 65-75% (чистая печь), до 82% (с водяной рубашкой) |
| Расход дров (дом B, 80 м2) | ~15-18 м3/сезон |
| Стоимость кладки | 80 000 - 250 000 руб |
| Площадь основания | 1-2 м2 |

**Плюсы**: Надёжность, долговечность, не требует электричества, традиционно, тепловой аккумулятор.
**Минусы**: Низкий КПД, большой расход дров, ручная закладка 3-4 раза/сутки, занимает площадь.

#### Вариант 5: Централизованная котельная на щепе/пеллетах

**Описание**: Одна или несколько котельных на кварталы 50-100 домов. Теплотрасса.

| Параметр | Значение |
|----------|----------|
| Мощность котельной | 500-1000 кВт (на квартал) |
| КПД промышленного котла | 88-93% |
| Теплотрасса | Предизолированные трубы (ПЭ-трубы, Изопрофлекс и т.п.) |
| Потери в теплотрассе | 10-15% |
| Общий КПД (с потерями) | 75-82% |
| Стоимость котельной (500 кВт) | 3-5 млн руб |
| Стоимость теплотрассы | 3 000 - 8 000 руб/м.п. (с работой) |

**Плюсы**: Профессиональное обслуживание, высокий КПД котла, удобство для жителей.
**Минусы**: Огромные капзатраты на теплотрассу, потери тепла при транспортировке, зависимость от центральной системы.

### 4.1.6 Расчёт расхода топлива на 1 дом за отопительный сезон

```
Расход = Q_сезон / (КПД * Теплота_сгорания_топлива)
```

### 4.1.8 Вывод по отоплению домов (консенсус специалистов)

**Рекомендация: КОМБИНИРОВАННЫЙ ПОДХОД**

1. **Базовая система** -- индивидуальный пиролизный (газогенераторный) котёл на дровах/брикетах с водяной системой отопления (радиаторы + тёплый пол на 1 этаже). Это даёт:
   - Минимальные капитальные затраты
   - Работу на местном топливе (дрова из леса, брикеты из отходов лесопилки)
   - Независимость каждого дома
   - КПД 82-88%

2. **Апгрейд для желающих** -- пеллетный автоматический котёл. Дороже, но комфортнее (загрузка раз в 3-7 дней).

3. **Резерв** -- каминная печь или кирпичная печь в каждом доме как аварийный источник тепла (не зависит от электричества для циркуляционного насоса).

4. **Централизованная котельная** -- НЕ рекомендуется для жилых домов на начальном этапе из-за:
   - Огромных затрат на теплотрассу (при 300 домах на 10 га -- средняя длина трассы ~2-3 км, стоимость 6-24 млн руб)
   - Потерь тепла 10-15%
   - Единой точки отказа
   - Но ЦЕЛЕСООБРАЗНА для кластера общественных зданий (школа + клуб + медпункт + баня)

## 4.2 ЦЕНТРАЛИЗОВАННАЯ ПИРОЛИЗНАЯ УСТАНОВКА (ТЭЦ)

### 4.2.1 Что такое пиролиз и зачем он поселению

**Пиролиз** -- термическое разложение органического вещества (древесины) при температуре 400-800 C без доступа кислорода (или с ограниченным доступом). Продукты пиролиза:

| Продукт | Выход (% от массы) | Применение |
|---------|---------------------|------------|
| Пиролизный газ (синтез-газ) | 15-25% | Привод газового двигателя-генератора, прямое сжигание |
| Жидкая фракция (пиролизная смола, конденсат) | 25-40% | Топливо, химическое сырьё |
| Древесный уголь (биочар) | 25-35% | Топливо, почвоулучшитель, сорбент |
| Тепло | 100% утилизируется | Отопление, нагрев воды |

**Газификация** -- близкий процесс, но при ограниченном доступе воздуха. Преимущество: почти весь углерод конвертируется в горючий газ (CO + H2 + CH4). Выход газа 70-80% энергии топлива.

### 4.2.2 Возможность выработки электричества из пиролизного/генераторного газа

**Ключевой вопрос: можно ли получить электричество из древесных отходов?**

**Ответ: ДА, это реальная и работающая технология.**

#### Схема когенерации (ТЭЦ на газификации древесины)

```
ЩЕПА/ОТХОДЫ → [ГАЗИФИКАТОР] → Синтез-газ → [ГАЗОВЫЙ ДВИГАТЕЛЬ] → ЭЛЕКТРИЧЕСТВО
                    ↓                              ↓
                  ТЕПЛО                        ТЕПЛО (охлаждение)
                    ↓                              ↓
              [УТИЛИЗАЦИЯ ТЕПЛА] ← ← ← ← ← ← ← ←
                    ↓
            ГОРЯЧАЯ ВОДА НА ОТОПЛЕНИЕ/ГВС
```

#### Характеристики синтез-газа из древесины

| Параметр | Значение |
|----------|----------|
| Состав (примерно) | CO 18-22%, H2 15-20%, CH4 2-4%, CO2 10-12%, N2 45-55% |
| Теплота сгорания | 4.5-6.0 МДж/нм3 (1.25-1.7 кВт*ч/нм3) |
| Выход газа из 1 кг древесины (влажн. 15-20%) | 2.0-2.5 нм3 |
| Энергетический выход | 2.5-4.0 кВт*ч газа из 1 кг древесины |

#### КПД когенерационной установки

| Компонент | КПД |
|-----------|-----|
| Газификатор (древесина → газ) | 70-80% |
| Газовый двигатель (газ → мех. энергия) | 25-32% |
| Генератор (мех. → электричество) | 92-95% |
| **Общий электрический КПД** | **16-24%** |
| Утилизация тепла двигателя и газификатора | 45-55% |
| **Общий КПД когенерации (эл + тепло)** | **65-80%** |

### 4.2.4 Расчёт пиролизной ТЭЦ для поселения

**Задача**: Обеспечить электроэнергией и теплом общественные здания + частично компенсировать электрические нужды домов.

**Выбранный вариант**: Газификационная ТЭЦ мощностью **200 кВт_эл / 400 кВт_тепл**

| Параметр | Значение |
|----------|----------|
| Режим работы | 7 000 ч/год (80% времени) |
| Выработка электроэнергии | **1 400 000 кВт*ч/год = 1 400 МВт*ч/год** |
| Выработка тепла (утилизация) | **2 800 000 кВт*ч/год = 2 800 МВт*ч/год** |
| Расход щепы (при 200 кг/ч) | **1 400 тонн/год = 5 600 м3 насыпной щепы** |
| Обслуживающий персонал | 2-3 оператора (посменно) |
| Потребление масла (двигатель) | ~5 л/сутки |
| Зола (отходы) | ~3% от массы сырья = 42 тонны/год (удобрение!) |

**Утилизация тепла**:
- Отопление кластера общественных зданий (школа + клуб + медпункт + баня + пекарня) -- ~500 МВт*ч/год
- Сушка щепы/пиломатериала -- ~500 МВт*ч/год
- Подогрев ближайших теплиц -- ~1 000 МВт*ч/год
- Горячая вода для бани/прачечной -- ~400 МВт*ч/год
- Потери -- ~400 МВт*ч/год

### 4.2.5 Альтернатива: Паровая турбина (ORC -- органический цикл Ренкина)

Вместо газификации + газовый двигатель можно использовать котёл на щепе + ORC-турбину.

| Параметр | Газификация + двигатель | Котёл + ORC |
|----------|------------------------|-------------|
| Электрический КПД | 18-24% | 10-16% |
| Тепловой КПД | 50-55% | 65-75% |
| Общий КПД | 70-78% | 80-88% |
| Сложность | Высокая (очистка газа) | Средняя |
| Стоимость (200 кВт_эл) | 10-20 млн руб | 15-30 млн руб |
| Обслуживание | Сложное | Среднее |
| Надёжность | Средняя | Высокая |

**Вывод**: Газификация даёт больше электричества, но сложнее. ORC проще и надёжнее, но дороже и менее эффективен по электрике. Для поселения **рекомендуется газификация** из-за приоритета электрогенерации.

## 4.4 ОТОПЛЕНИЕ ТЕПЛИЦ

### 4.4.1 Теплотехнический расчёт теплиц

**Базовая теплица**: двойной поликарбонат 8 мм, арочная, размер 6 x 30 м = 180 м2

| Параметр | Значение |
|----------|----------|
| Площадь ограждающих конструкций (стены + крыша) | ~260 м2 |
| R поликарбоната 8 мм (двойной) | 0.4-0.5 м2*C/Вт |
| Площадь грунта | 140 м2 (полезная) |
| T внутри зимой (минимум) | +12...+15 C (для зелени и холодостойких), +18...+22 C (для томатов/перцев) |
| T наружная | -30 C (расчётная) |
| Дельта T | 42-52 C |

**Теплопотери 1 теплицы при -30 C, T_внутри = +15 C (дельта = 45 C)**:

```
Q_огр = 260 м2 / 0.45 м2*C/Вт * 45 C = 26 000 Вт = 26 кВт
Q_инф = 30% * Q_огр = 7.8 кВт
Q_грунт = ~3 кВт
ИТОГО: ~37 кВт на 1 теплицу 180 м2
```

**Удельная теплопотеря: ~205 Вт/м2 площади теплицы при -30 C**

Для поддержания +22 C (дельта 52 C): ~240 Вт/м2, т.е. ~43 кВт на теплицу.

### 4.4.2 Потребности в тепле для тепличного хозяйства

| Сценарий | Площадь | Q_пик (-30 C), кВт | Q_средний за сезон*, кВт | Q_сезон, МВт*ч |
|----------|---------|---------------------|--------------------------|----------------|
| 1-я очередь | 2 000 м2 (~11 теплиц) | 410 | 180 | 910 |
| Полный объём | 5 000 м2 (~28 теплиц) | 1 025 | 450 | 2 270 |

*Средний = пиковый * (45 - 3.5 - 0) / 45 = ~0.44 (с учётом, что теплицы работают круглогодично, но летом не топятся)

Уточнение: теплицы отапливаются ~240 дней (с середины сентября по середину мая), средняя дельта T = 20 C

Q_средний = Q_пик * 20/45 = 0.44 * Q_пик

### 4.4.3 Варианты отопления теплиц

#### Вариант 1: Утилизация тепла газификационной ТЭЦ

**Лучший вариант**, если теплицы рядом с ТЭЦ (< 100 м).

| Параметр | Значение |
|----------|----------|
| Доступное тепло от ТЭЦ (200 кВт_эл) | 400 кВт_тепл |
| Покрытие потребности (1-я очередь, 2000 м2) | ~100% средней нагрузки, ~97% пиковой |
| Дополнительные затраты | Теплотрасса 50-100 м, теплообменники, регистры в теплицах |
| Стоимость | 1-2 млн руб |
| Эксплуатационные расходы | Минимальные (тепло -- побочный продукт) |

**Эффект**: Теплицы получают "бесплатное" тепло, повышая общий КПД ТЭЦ до 75-80%.

#### Вариант 2: Отдельная котельная на щепе/дровах

| Параметр | Значение |
|----------|----------|
| Мощность котла | 200-500 кВт |
| Расход щепы (на 2000 м2 теплиц) | ~400 тонн/сезон |
| Стоимость котла | 1-3 млн руб |
| Стоимость системы | 2-5 млн руб (с обвязкой) |

#### Вариант 3: Индивидуальные печи в каждой теплице

| Параметр | Значение |
|----------|----------|
| Тип | Буллерьян / печь длительного горения |
| Мощность | 15-30 кВт |
| Расход дров на 1 теплицу | ~15-20 м3/сезон |
| Стоимость печи | 30 000-80 000 руб |

**Плюсы**: Простота, дешевизна, автономность каждой теплицы.
**Минусы**: Ручная загрузка 2-3 раза/сутки, неравномерный прогрев, пожароопасность.

#### Вариант 4: Биоразогрев грунта (вспомогательный)

Закладка навоза/компоста под грядки. При разложении выделяется тепло до +25 C в толще грунта.

| Параметр | Значение |
|----------|----------|
| Мощность | ~20-30 Вт/м2 (небольшая) |
| Длительность эффекта | 2-3 месяца |
| Расход навоза | 10-15 кг/м2 |

**Вердикт**: Дополнительная мера, не заменяет основное отопление при -30 C.

#### Вариант 5: Подземные (заглублённые) теплицы -- walipini

Грунт ниже глубины промерзания (+4...+8 C) служит "аккумулятором тепла".

| Параметр | Значение |
|----------|----------|
| Заглубление | 1.5-2.5 м |
| Экономия на отоплении | 30-50% |
| Стоимость строительства | В 2-3 раза выше наземной |
| Проблемы | Дренаж, освещение, вентиляция |

**Вердикт**: Перспективно для малых теплиц (зелень, рассада). Для промышленных объёмов -- дорого и сложно.

### 4.4.4 Рекомендация по отоплению теплиц

1. **Размещать теплицы рядом с пиролизной ТЭЦ** (в радиусе 100-200 м)
2. **Основное тепло** -- утилизация от ТЭЦ через водяной контур
3. **Резерв** -- котёл на щепе 200 кВт в здании ТЭЦ (на случай пиковых морозов или остановки ТЭЦ)
4. **Дополнительно** -- в каждой теплице печь-буржуйка как аварийный источник
5. **Двойной поликарбонат** -- ОБЯЗАТЕЛЬНО для зимних теплиц (снижает потери на 30-40%)
6. **Тепловые завесы** -- над входами
7. **Тёплые грядки с биоразогревом** -- дополнительно

## 7. РЕКОМЕНДУЕМАЯ ЭНЕРГЕТИЧЕСКАЯ СТРАТЕГИЯ

## ПРИЛОЖЕНИЕ A: КЛЮЧЕВЫЕ ВЫВОДЫ МОЗГОВОГО ШТУРМА (ИТЕРАЦИИ 1-20)

### Итерации 1-4: Определение масштабов

**Теплотехник**: Базовый расчёт показал: при нормальном утеплении (200 мм минвата) дом 80 м2 теряет ~5.3 кВт при -30 C. Это не много. Проблема не в пиковой мощности, а в объёмах топлива на 300 домов за 210 дней.

**Экономист**: 5 700 МВт*ч тепла в год -- это серьёзный объём. При пересчёте в дрова -- 3 500 м3. При средней цене дров 1 500 руб/м3 -- это 5.25 млн руб только на отопление домов. Но если своя заготовка -- в 2-3 раза дешевле.

**Специалист по биомассе**: 3 500 м3 колотых дров = ~6 500 м3 кругляка (с учётом потерь при распиловке/колке). Это ~32 га леса при среднем запасе 200 м3/га. Устойчивая ротация требует 1 600 га леса в обороте (50-летний цикл). Это реалистично для средней полосы -- леса много.

### Итерации 5-8: Пиролиз и когенерация

**Специалист по пиролизу**: Электричество из древесины -- реальность. Газификация древесной щепы + газопоршневой двигатель даёт 18-24% электрического КПД. При 200 кВт_эл на выходе получаем 400 кВт тепла -- идеально для теплиц и общественных зданий. Критически важна чистота щепы (без земли, металла) и её влажность (< 25%).

**Энергетик**: 200 кВт_эл покрывает ~70-80% электропотребления поселения в среднем. Но пиковая нагрузка зимним вечером -- 500-600 кВт. Нужен либо второй блок ТЭЦ, либо сетевое подключение как резерв. Дизель -- только на аварию. Рекомендую конфигурацию "ТЭЦ 200 кВт + сеть 200 кВт" или "2 x ТЭЦ 200 кВт + дизель 150 кВт".

**Экономист**: Себестоимость электричества от газификационной ТЭЦ на своей щепе: 1.5-3.5 руб/кВт*ч (включая амортизацию). Тариф сети: 5-10 руб/кВт*ч. Экономия: 3-7 руб/кВт*ч * 1 600 МВт*ч = 4.8-11.2 млн руб/год. Окупаемость ТЭЦ: 3-5 лет. Это ОТЛИЧНЫЙ проект.

### Итерации 9-12: Индивидуальное vs централизованное отопление

**Теплотехник**: Центральная котельная на 300 домов -- это 2 МВт мощности. Котёл такой мощности стоит 3-5 млн руб, что дёшево. НО теплотрасса -- 3-5 км при разбросанной застройке. Это 15-40 млн руб. Плюс потери 10-15%. Плюс единая точка отказа. Для 300 домов на 10 га это не оптимально.

**Специалист по биомассе**: Индивидуальные котлы проще в топливной логистике: каждый дом имеет свой дровник, хозяин сам управляет запасом. При централизации нужен механизированный склад на 10 000+ м3 щепы, что дорого и сложно.

**Экономист**: Сравнил CAPEX: индивидуальные пиролизные котлы на 300 домов = 54-105 млн руб. Центральная котельная + теплотрасса = 30-80 млн руб. Разница невелика, но индивидуальные системы дают автономность. А при поломке одного котла замерзает один дом, не всё поселение.

**Консенсус**: Индивидуальное отопление для жилых домов. Центральная мини-котельная -- только для кластера общественных зданий и теплиц (от ТЭЦ).

### Итерации 13-16: Тепличное хозяйство и отходы лесопилки

**Теплотехник**: Теплица -- самый проблемный потребитель тепла. При -30 C теплица 180 м2 теряет 37 кВт. 28 теплиц (5 000 м2) -- это 1 025 кВт пиковой нагрузки. Это БОЛЬШЕ, чем тепло от ТЭЦ 400 кВт. Решение: (а) двойной поликарбонат (-30% потерь), (б) тепловые экраны на ночь, (в) заглублённые теплицы для зелени, (г) резервный котёл.

**Специалист по биомассе**: Лесопилка перерабатывает 5 000-10 000 м3 кругляка. Отходы -- 2 000-5 000 м3 (800-2 000 тонн). ТЭЦ потребляет 2 800 тонн щепы. Дефицит -- 800-2 000 тонн, который покрывается дополнительной заготовкой низкосортной древесины (тонкомер, вершины, сухостой). Это идеальная синергия: лесопилка даёт отходы для ТЭЦ, ТЭЦ даёт тепло для сушки пиломатериала и электричество для станков.

**Специалист по пиролизу**: Предлагаю дополнительно рассмотреть производство древесного угля как товарного продукта. Если часть отходов (500 тонн/год) направить на углежжение -- получим ~150 тонн древесного угля. При цене 30-40 тыс руб/тонна -- это 4.5-6 млн руб дохода. Тепло от процесса тоже утилизируется.

### Итерации 17-20: Экономика, фазирование, риски

**Экономист**: Общая стоимость энергетической инфраструктуры: 136-268 млн руб. На 300 домов это 450-890 тыс руб/дом. Годовые расходы: 14-32 млн руб = 47-107 тыс руб/дом/год (4-9 тыс руб/мес). Это ЗНАЧИТЕЛЬНО дешевле городского жилья, где одно отопление + электричество = 5-15 тыс руб/мес.

**Все специалисты -- риски**:
1. **Влажность щепы** -- если подавать мокрую щепу в газификатор, КПД падает на 20-30%, возможны аварии. КРИТИЧЕСКИ ВАЖНА сушка.
2. **Квалификация персонала** -- газификационная ТЭЦ требует обученных операторов. Минимум 2-3 человека.
3. **Запас топлива** -- обязательный аварийный запас на 2-3 недели автономной работы без подвоза.
4. **Электрозависимость** -- циркуляционные насосы в домах зависят от электричества. При отключении ТЭЦ -- отопление встаёт. Решение: (а) ИБП в каждом доме (15-20 тыс руб), (б) резервный генератор, (в) кирпичная печь как аварийный источник.

**Энергетик**: При выборе участка ОБЯЗАТЕЛЬНО проверить:
- Расстояние до ближайшей ЛЭП 10 кВ (< 3 км -- подключаться)
- Наличие реки/ручья (микро-ГЭС -- бонус)
- Наличие леса в радиусе 20 км (топливная база)
- Рельеф (для гравитационной подачи воды, если возможно)

**Компоненты Smart Grid**:

| Компонент | Кол-во | Назначение | Стоимость ед. | Итого |
|-----------|--------|------------|---------------|-------|
| Умные счётчики (Меркурий 236, Нева МТ 314) | 300 | Учёт потребления, телеметрия, дистанционное отключение | 5 000-12 000 руб | 1.5-3.6 млн руб |
| Контроллер нагрузки (Arduino/Raspberry + реле) | 300 | Управление неприоритетными нагрузками в доме | 3 000-7 000 руб | 0.9-2.1 млн руб |
| Концентратор данных (на квартал) | 3-4 | Сбор данных от 100 счётчиков, передача на сервер | 20 000-50 000 руб | 60-200 тыс руб |
| Линия связи (витая пара / RS-485 / PLC) | 5-8 км | Передача данных | 50-150 руб/м.п. | 250-1 200 тыс руб |
| Сервер SCADA + ПО | 1 | Мониторинг, управление, журналирование | 200 000-500 000 руб | 200-500 тыс руб |
| Автоматические выключатели с дистанционным управлением | 4-6 | Секционирование фидеров, локализация аварий | 30 000-80 000 руб | 120-480 тыс руб |
| **ИТОГО Smart Grid** | | | | **3-8 млн руб** |

**Функции Smart Grid**:

1. **Управление нагрузкой (Demand Response)**:
   - При приближении к лимиту мощности ТЭЦ (>90% нагрузки) — автоматическое отключение неприоритетных нагрузок: бойлеры ГВС, стиральные машины, электроплиты (с уведомлением жителей по SMS/приложению)
   - Экономия в пике: 80-120 кВт (20-30% пиковой нагрузки)

2. **Балансировка между фидерами**:
   - Если один квартал потребляет меньше нормы, избыток перебрасывается на другой
   - Предотвращает перегрузку отдельных линий

3. **Мониторинг в реальном времени**:
   - Потребление каждого дома
   - Генерация ТЭЦ
   - Состояние сети (напряжение, частота, cos φ)
   - Прогноз нагрузки на 24 ч (по данным температуры и исторических паттернов)

4. **Аварийное управление**:
   - При отказе одного блока ТЭЦ — автоматическое ограничение нагрузки до мощности оставшегося блока
   - Приоритетное питание: медпункт → школа → общественный центр → жилые дома (по ротации)

**Экономист энергетики**: Инвестиции 3-8 млн руб в Smart Grid окупаются за счёт:
- Снижение пиковой нагрузки на 20-30% → можно не строить второй блок ТЭЦ (экономия 10-15 млн руб)
- Предотвращение аварий и перегрузок → снижение потерь от простоев
- Оптимизация расхода топлива через выравнивание графика нагрузки → экономия 5-10% топлива = 200-500 тыс руб/год

### Итерация 23: Системы накопления энергии (аккумуляторы)

**Инженер-электрик**: Рассмотрим возможность установки аккумуляторных батарей для сглаживания пиков и резервирования. Вопрос — насколько это реалистично в российских климатических условиях.

**Задачи аккумуляторной системы**:
1. Покрытие вечернего пика (3-4 часа, дефицит ~100-200 кВт при одном блоке ТЭЦ)
2. Резерв на время обслуживания ТЭЦ (4-8 часов)
3. Интеграция солнечных панелей (если установлены) — накопление дневной выработки

**Расчёт ёмкости**:

| Сценарий | Мощность, кВт | Время, ч | Ёмкость, кВт*ч | Тип батареи |
|----------|---------------|----------|-----------------|-------------|
| Сглаживание пика | 150 | 4 | 600 | Li-ion (LiFePO4) |
| Аварийный резерв (критические потребители) | 50 | 8 | 400 | Li-ion (LiFePO4) |
| Полный резерв (всё поселение) | 275 | 8 | 2 200 | Нереалистично по стоимости |

**Сравнение технологий аккумуляторов**:

| Параметр | LiFePO4 (литий-железо-фосфат) | AGM свинцово-кислотные | OPzS (тяговые) | Ванадиевые проточные |
|----------|-------------------------------|------------------------|----------------|---------------------|
| Удельная энергия | 90-120 Вт*ч/кг | 30-40 Вт*ч/кг | 25-35 Вт*ч/кг | 15-25 Вт*ч/кг |
| Циклический ресурс | 3 000-6 000 циклов | 500-800 циклов | 1 500-2 500 циклов | 10 000+ циклов |
| Рабочая температура | -20...+60 C | -15...+50 C | +5...+45 C | +10...+40 C |
| Стоимость за 1 кВт*ч ёмкости | 20 000-35 000 руб | 8 000-15 000 руб | 12 000-20 000 руб | 40 000-80 000 руб |
| Срок службы | 10-15 лет | 3-5 лет | 7-12 лет | 15-20 лет |
| LCOE хранения (за цикл) | 3-6 руб/кВт*ч | 10-20 руб/кВт*ч | 5-10 руб/кВт*ч | 4-8 руб/кВт*ч |

**Специалист по ВИЭ**: Критический фактор для России — температура. При -30 C:
- LiFePO4: ёмкость падает на 20-30%, заряд ниже -10 C запрещён (деградация анода). Нужен отапливаемый контейнер.
- Свинцовые: ёмкость падает на 40-50%, электролит может замёрзнуть при -35 C.
- Ванадиевые: электролит замерзает при -10 C — однозначно в тёплом помещении.

**Вывод**: Любые аккумуляторы в климате средней полосы России требуют отапливаемого помещения (контейнер с утеплением + обогрев от ТЭЦ).

**Стоимость аккумуляторной системы 600 кВт*ч (покрытие пика)**:

| Компонент | Стоимость |
|-----------|-----------|
| Батареи LiFePO4 600 кВт*ч (с запасом 20% = 720 кВт*ч) | 14.4-25.2 млн руб |
| Инвертор-зарядник 200 кВт (2 x 100 кВт) | 1.5-3 млн руб |
| Контейнер утеплённый 20 футов | 300-600 тыс руб |
| Обогрев контейнера (2 кВт) | 50-100 тыс руб |
| BMS (система управления батареями) | 200-500 тыс руб |
| Монтаж и пуско-наладка | 500 тыс - 1 млн руб |
| **ИТОГО** | **17-30 млн руб** |

**Экономист энергетики**: При стоимости 17-30 млн руб за 600 кВт*ч — это 28 000-50 000 руб/кВт*ч установленной ёмкости. Экономический эффект:
- Экономия дизельного топлива при аварийных запусках: ~200-400 тыс руб/год
- Снижение пиковой нагрузки ТЭЦ: продление ресурса двигателя, экономия масла — 100-200 тыс руб/год
- **Срок окупаемости: 30-60 лет** — экономически НЕ оправдано как самостоятельный проект

**Рекомендация**: Аккумуляторная система в полном объёме (600 кВт*ч) на начальном этапе НЕ рекомендуется. Целесообразно:
1. **Минимальный резерв**: 50-100 кВт*ч LiFePO4 для критических потребителей (медпункт, серверная, аварийное освещение) — **1.5-4 млн руб**
2. **ИБП в каждом доме**: 1-2 кВт*ч для циркуляционного насоса (2-4 часа автономности) — **15 000-25 000 руб/дом**, итого **4.5-7.5 млн руб на 300 домов**
3. **Пересмотр решения через 5-7 лет**: цены на LiFePO4 снижаются на 10-15% в год, к 2032-2035 году полная система может стать рентабельной

### Итерация 24: Грунтовые тепловые насосы в условиях -30 C

**Инженер теплоснабжения**: Тепловые насосы (ТН) — технология, которую нельзя игнорировать. Рассмотрим грунтовые (ground-source) ТН для нашего климата. Воздушные ТН при -30 C практически неработоспособны (COP < 1.5), поэтому рассматриваем только грунтовые.

**Принцип**: Температура грунта ниже глубины промерзания (~1.5-1.8 м для средней полосы) стабильна: **+5...+8 C** круглый год. Грунтовый ТН извлекает это тепло и повышает до +35...+55 C для отопления.

**Параметры грунтового ТН для дома 80 м2 (тип B)**:

| Параметр | Значение |
|----------|----------|
| Теплопотери дома при -30 C | 5.3 кВт |
| Требуемая мощность ТН | 6-8 кВт (тепловых) |
| COP при t_грунта = +5 C, t_подачи = +35 C (тёплый пол) | **3.5-4.0** |
| COP при t_грунта = +5 C, t_подачи = +55 C (радиаторы) | **2.5-3.0** |
| Электрическая мощность компрессора (при COP 3.5, Q = 6 кВт) | **1.7 кВт** |
| Длина грунтового контура (горизонтальный) | 300-500 м.п. на глубине 1.5-2 м |
| Площадь участка для горизонтального контура | **150-250 м2** (дополнительно к дому) |
| Длина скважин (вертикальный контур) | 2 скважины x 50-70 м = 100-140 м.п. |

**Стоимость грунтового ТН на 1 дом**:

| Компонент | Горизонтальный контур | Вертикальный контур |
|-----------|----------------------|---------------------|
| Тепловой насос 8 кВт (Nibe, Vaillant, российский Корса) | 200 000-500 000 руб | 200 000-500 000 руб |
| Земляные работы + трубы контура | 80 000-150 000 руб | 200 000-400 000 руб (бурение) |
| Теплоноситель (пропиленгликоль 30%) | 15 000-25 000 руб | 15 000-25 000 руб |
| Система отопления (тёплый пол — ОБЯЗАТЕЛЬНО для низкотемпературного контура) | 120 000-200 000 руб | 120 000-200 000 руб |
| Буферная ёмкость 200 л | 20 000-40 000 руб | 20 000-40 000 руб |
| Монтаж и пуско-наладка | 50 000-100 000 руб | 50 000-100 000 руб |
| **ИТОГО на 1 дом** | **485 000-1 015 000 руб** | **605 000-1 265 000 руб** |
| **ИТОГО на 300 домов** | **145-305 млн руб** | **182-380 млн руб** |

**Специалист по ВИЭ**: Главная проблема грунтовых ТН в нашем проекте — электрозависимость. При COP 3.5 на 300 домов:
- Средняя тепловая нагрузка за сезон: 764 кВт (отопление домов)
- Электрическая мощность для ТН: 764 / 3.5 = **218 кВт** — это ВДВОЕ увеличивает пиковую электрическую нагрузку!
- Годовое потребление электричества: 218 кВт * 5 040 ч (сезон) = **1 098 МВт*ч** — это удвоение текущего электропотребления

**Экономист энергетики**: Сравним три сценария для 300 домов:

| Параметр | Пиролизный котёл (дрова) | Грунтовый ТН (горизонт.) | Грунтовый ТН (вертик.) |
|----------|--------------------------|--------------------------|------------------------|
| CAPEX на 300 домов | 54-105 млн руб | 145-305 млн руб | 182-380 млн руб |
| Годовое топливо/электричество | 2-4 млн руб (дрова свои) | 4.4-7.7 млн руб (электричество) | 4.4-7.7 млн руб |
| Обслуживание | 1-2 млн руб/год | 0.5-1 млн руб/год | 0.5-1 млн руб/год |
| Комфорт пользователя | Загрузка 1-2 р/сутки | Полный автомат | Полный автомат |
| Зависимость от электричества | Минимальная (насос 50 Вт) | Полная (1.7 кВт/дом) | Полная |
| Срок службы | 15-20 лет (котёл) | 20-25 лет (ТН), 50 лет (контур) | 25 лет (ТН), 50+ лет (скважины) |

**Инженер теплоснабжения**: Ключевая проблема — при отключении электричества дом с ТН мгновенно теряет отопление. При -30 C каркасный дом остывает до критических +5 C за 8-12 часов. С пиролизным котлом — можно топить при свечах.

**Вывод консенсусом**: Грунтовые тепловые насосы на текущем этапе **НЕ рекомендуются** как массовое решение для поселения из-за:
1. Капзатраты в 2-3 раза выше пиролизных котлов
2. Критическая зависимость от электросети (необходимо удвоить мощность генерации)
3. Философия поселения — автономность, а ТН привязывают к электросети

**Где ТН целесообразны**: Общественные здания (школа, медпункт) с гарантированным электроснабжением от ТЭЦ — здесь COP 3.5 снижает расход топлива на 65%. Для 2-3 зданий общей площадью 1 000 м2 можно установить промышленный ТН 30-50 кВт_тепл за 1-2 млн руб.

### Итерация 26: Электробезопасность и система заземления поселения

**Инженер-электрик**: Тема критически важная и часто упускаемая при проектировании автономных поселений. Рассмотрим три аспекта: система заземления, защитное отключение, молниезащита.

**26.1 Система заземления**

Для поселения с собственной генерацией (ТЭЦ) применяется система **TN-C-S** (по ГОСТ Р 50571):
- На ТЭЦ/ТП: нейтраль генератора заземлена через контур заземления (TN-C)
- На вводе в дом: разделение PEN → PE + N (переход в TN-S)
- В доме: раздельные проводники N (нулевой рабочий) и PE (нулевой защитный)

**Контур заземления ТЭЦ/ТП**:

| Параметр | Значение |
|----------|----------|
| Тип | Горизонтальные и вертикальные электроды в грунте |

### Итерация 27: Коррекция коэффициента мощности и качество электроэнергии

**Инженер-электрик**: При наличии собственной газификационной ТЭЦ качество электроэнергии — ответственность самого поселения. Рассмотрим ключевые проблемы.

**27.1 Коэффициент мощности (cos φ)**

Проблема: электродвигатели (лесопилка, насосы, компрессоры, вентиляторы) потребляют реактивную мощность, снижая cos φ. При cos φ = 0.7 генератор ТЭЦ на 200 кВт отдаёт только 140 кВт активной мощности.

**Оценка реактивной нагрузки поселения**:

| Потребитель | Активная P, кВт | cos φ | Реактивная Q, кВАр |
|-------------|-----------------|-------|---------------------|
| Жилые дома (освещение LED, электроника) | 115 | 0.95 | 37 |
| Лесопилка (электродвигатели) | 50 | 0.70 | 51 |
| Столярная мастерская | 20 | 0.75 | 18 |
| Насосы водоснабжения/канализации | 15 | 0.80 | 11 |
| Вентиляция теплиц | 10 | 0.78 | 8 |
| Холодильные камеры | 10 | 0.75 | 9 |
| Сварочные аппараты (эпизодически) | 20 | 0.60 | 27 |
| **ИТОГО** | **240** | **0.82 (средневзв.)** | **161** |

**Полная мощность генератора**: S = P / cos φ = 240 / 0.82 = **293 кВА**

При cos φ = 0.82 генератор 200 кВт (250 кВА) загружен на 293/250 = 117% — **ПЕРЕГРУЗКА**!

**Решение: установка конденсаторных батарей (КРМ — компенсация реактивной мощности)**

| Параметр | Значение |
|----------|----------|
| Целевой cos φ | 0.95 |
| Требуемая компенсация | Q_комп = P * (tg φ₁ - tg φ₂) = 240 * (0.70 - 0.33) = **89 кВАр** |
| Тип КРМ | Автоматическая конденсаторная установка (АКУ) со ступенчатым регулированием |
| Кол-во ступеней | 5-7 (по 12.5-20 кВАр) |
| Стоимость АКУ 100 кВАр | **150 000-350 000 руб** |
| Место установки | Главный распределительный щит ТЭЦ |

**Эффект компенсации**:

| Параметр | До компенсации | После компенсации |
|----------|----------------|-------------------|
| cos φ | 0.82 | 0.95 |
| Полная мощность | 293 кВА | 253 кВА |
| Загрузка генератора (250 кВА) | 117% (перегрузка!) | 101% (норма) |
| Потери в кабелях (∝ I²) | 100% (базовый) | 74% (-26%) |
| Экономия топлива ТЭЦ | — | 3-5% |

**Дополнительно: локальная компенсация на производстве**:

| Объект | КРМ | Стоимость |
|--------|-----|-----------|
| Лесопилка (крупные двигатели 15-30 кВт) | КРМ 30 кВАр (индивидуальная) | 60 000-120 000 руб |
| Столярная мастерская | КРМ 15 кВАр | 40 000-80 000 руб |
| **ИТОГО локальная КРМ** | | **100 000-200 000 руб** |

**27.2 Частота и напряжение**

При автономной генерации (без привязки к внешней сети) частота и напряжение определяются газопоршневым двигателем ТЭЦ:

| Параметр | Норматив (ГОСТ 32144-2013) | Достижимое на ТЭЦ |
|----------|----------------------------|---------------------|
| Частота | 50 Гц ± 0.2 Гц (нормальный режим) | 50 Гц ± 0.5-1.0 Гц |
| Напряжение | 220 В ± 10% (198-242 В) | 220 В ± 5-7% |
| Коэффициент несинусоидальности (THD) | ≤ 8% | 5-10% (ГПД) |

**Проблемы и решения**:

| Проблема | Причина | Решение | Стоимость |
|----------|---------|---------|-----------|
| Провалы частоты при набросе нагрузки | Медленная реакция газопоршневого двигателя (1-3 сек) | Маховик на валу + быстродействующий регулятор оборотов | Входит в стоимость ТЭЦ |
| Скачки напряжения при пуске моторов | Пусковой ток двигателя 5-7 x In | Устройства плавного пуска (УПП) на всех двигателях >5 кВт | 10 000-50 000 руб/шт |
| Гармоники от преобразователей частоты, ИБП, LED | Нелинейные нагрузки | Пассивные фильтры гармоник | 50 000-150 000 руб |

**Экономист энергетики**: Суммарные затраты на обеспечение качества электроэнергии:

| Компонент | Стоимость |
|-----------|-----------|
| АКУ 100 кВАр (главная) | 150 000-350 000 руб |
| Локальная КРМ (производство) | 100 000-200 000 руб |
| УПП для двигателей (8-10 шт) | 80 000-500 000 руб |
| Фильтры гармоник | 50 000-150 000 руб |
| **ИТОГО** | **380 000-1 200 000 руб** |

Невелика сумма, но критически важна для здоровья генератора и электрооборудования. Без КРМ генератор перегружается, перегревается, срок службы сокращается с 40 000 до 15 000 моточасов.

**Специалист по ВИЭ**: Солнечные коллекторы (не путать с фотоэлектрическими панелями!) для нагрева воды — гораздо эффективнее в нашем климате, чем солнечные батареи для электричества. КПД коллектора 40-70% vs 15-20% у фотоэлектрики.

**Вакуумный трубчатый коллектор** — оптимальный выбор для холодного климата (работает эффективно до -30 C):

| Параметр | Значение |
|----------|----------|
| Площадь 1 коллектора | 2-3 м2 |
| Производительность (апрель-сентябрь) | 3-5 кВт*ч/(м2*день) |
| Производительность (октябрь-март) | 0.5-2 кВт*ч/(м2*день) |
| Среднегодовая производительность | 600-900 кВт*ч/(м2*год) |
| Стоимость 1 м2 коллектора (с монтажом) | 8 000-15 000 руб |
| Срок службы | 15-25 лет |

**Расчёт для 1 дома (семья 4 чел)**:

| Параметр | Значение |
|----------|----------|
| Потребность ГВС | 11 кВт*ч/сутки (60 C) |

**Экономист энергетики**: Окупаемость 15-30 лет — неприемлема для индивидуальных домов. Но есть сценарий, где солнечная термальная система имеет смысл:

**Централизованная солнечная термальная система для общественной бани**:

**Инженер теплоснабжения**: Всё ещё долгая окупаемость. Мой вердикт:
- Солнечная термальная система — **НЕ ПРИОРИТЕТ** на начальных этапах
- Целесообразна как опция для индивидуальных домов (за счёт жителей)
- Для общественных объектов — только если есть целевой грант или субсидия
- Лучший вариант летнего ГВС: электрический бойлер 2-3 кВт от ТЭЦ (тариф 1.5-3.5 руб/кВт*ч) — дешевле и проще

### Итерация 30: Мониторинг, управление и дорожная карта CO2

**30.1 Система энергетического мониторинга**

**Инженер-электрик**: Без мониторинга невозможно оптимизировать энергосистему. Проектируем Energy Management System (EMS).

**Архитектура EMS SLOBODA**:

```
[Датчики и счётчики]
    ├── Электросчётчики (300 домов + производство + инфра)
    ├── Теплосчётчики (ТЭЦ → теплицы, ТЭЦ → общ. здания)
    ├── Расходомеры топлива (щепа — весовой контроль, дизель — расходомер)
    ├── Термодатчики (наружная T, T подачи/обратки теплоносителя)
    ├── Датчики тока и напряжения (на фидерах)
    └── Метеостанция (T, влажность, ветер, инсоляция)
         |
    [Контроллеры сбора данных]
    ├── PLC на ТЭЦ (Siemens S7-1200 / Овен ПЛК210)
    ├── Концентраторы кварталов (Raspberry Pi + RS-485)
    └── IoT-шлюзы (Modbus → MQTT/HTTP)
         |
    [Сервер EMS]
    ├── СУБД (TimescaleDB / InfluxDB) — хранение временных рядов
    ├── SCADA (OpenSCADA / Мастер SCADA) — визуализация, управление
    ├── Аналитический модуль — отчёты, прогнозы, аномалии
    └── Веб-интерфейс — доступ оператора и жителей (личный кабинет)
```

| Потребление | Электрическая нагрузка по фидерам и домам | 15 мин | Счётчики |

**30.2 Расчёт и дорожная карта снижения CO2-выбросов**

**Эксперт по пиролизу и биомассе**: Расчёт углеродного следа поселения.

**Выбросы CO2 по источникам**:

*Древесное топливо считается углеродно-нейтральным при условии восполнения лесных запасов. При сжигании 4 800 тонн древесины в атмосферу выбрасывается ~8 160 тонн CO2, но растущий лес поглощает столько же за 50-летний цикл ротации.

**При условии**: лесовосстановление на вырубленных делянках ≥ 22 га/год. Необходимо посадить ~44 000 саженцев/год (2 000 шт/га).

**Сравнение с «традиционным» энергоснабжением**:

| Сценарий | CO2, тонн/год | Разница |
|----------|---------------|---------|

| Аналогичный посёлок на газе (магистральном) | ~1 800 | +1 643 (+1046%) |

**Экономист энергетики**: Поселение генерирует в 11-20 раз меньше ископаемого CO2, чем стандартный населённый пункт аналогичного размера. Это серьёзный аргумент для привлечения грантов, ESG-финансирования и углеродных кредитов.

**Дорожная карта снижения CO2 до нуля (net-zero)**:

| Этап | Действие | Снижение CO2, тонн/год | Стоимость | Срок |
|------|----------|------------------------|-----------|------|
| 1 | Перевод лесовозной техники на газ (КПГ из синтез-газа ТЭЦ) | -20 | 2-4 млн руб | 2030-2032 |
| 2 | Электрические бензопилы + электротрактор (заряд от ТЭЦ) | -30 | 3-5 млн руб | 2032-2034 |
| 3 | Отказ от резервного электричества из сети (2-й блок ТЭЦ + аккумуляторы) | -80 | 15-25 млн руб | 2033-2035 |
| 4 | Замена дизельного аварийного генератора на аккумуляторы | -13 | 5-10 млн руб | 2035-2037 |
| 5 | Полное лесовосстановление (посадка саженцев = поглощение) | Компенсация биомассы | 200-400 тыс руб/год | Непрерывно |
| **Итого Net-Zero** | | **~0 тонн CO2** | **25-44 млн руб** | **К 2037** |

**Эксперт по пиролизу и биомассе**: Дополнительная возможность — **биочар (древесный уголь) как средство секвестрации углерода**. При пиролизе часть углерода фиксируется в угле, который стабилен в почве сотни лет. Если 10% щепы ТЭЦ (280 тонн) направить на углежжение:
- Выход биочара: ~80 тонн/год
- Содержание углерода в биочаре: ~80% = 64 тонны C = **235 тонн CO2-эквивалента секвестрировано**
- Это делает поселение **углеродно-отрицательным** (negative carbon) — оно поглощает больше CO2, чем выбрасывает!

**Инженер теплоснабжения**: При внесении биочара в тепличные грядки достигается двойной эффект: секвестрация углерода + улучшение почвы (повышение влагоёмкости, катионообменной ёмкости, снижение кислотности). Расход 1-5 кг/м2 грядок. На 5 000 м2 теплиц = 5-25 тонн/год — часть продукции углежжения.

**Экономист энергетики**: Углеродно-отрицательный статус поселения — уникальное преимущество. На добровольном углеродном рынке (VCM) 1 тонна CO2 стоит $5-50 (2026 год). При чистой секвестрации 80 тонн CO2/год это $400-4 000 — символическая сумма, но по мере роста цены на углерод (прогноз $50-150 к 2035) доход может составить $12 000-30 000/год. Пока это нерелевантно экономически, но стратегически важно позиционирование.

## ПРИЛОЖЕНИЕ B: КРАТКИЕ ОТВЕТЫ НА КЛЮЧЕВЫЕ ВОПРОСЫ

### 1. Можно ли из пиролизного газа получить электричество? Какой мощности?

**Да.** Технология газификации древесины + газопоршневой двигатель позволяет получить 18-24% электрической энергии от сжигаемой биомассы. Реалистичная мощность для поселения: 100-400 кВт электрических. При этом попутно генерируется 200-800 кВт тепла. Оборудование производится в России (Энергоинтел, Казань) и за рубежом (Spanner, Burkhardt). Стоимость: 5-35 млн руб в зависимости от мощности.

### 2. Сколько древесины/биомассы нужно на 300-500 домов + теплицы в год?

**4 000-5 000 тонн** древесного топлива в год (для 300 домов). Это эквивалент ~10 000-12 000 м3 кругляка. Из них:
- ~2 000 тонн дров для отопления домов (или 1 400 тонн пеллет)
- ~2 800 тонн щепы для ТЭЦ
- ~200-300 тонн на прочие нужды (баня, пекарня, резерв)

Для 500 домов: ~6 500-8 000 тонн/год.

### 3. Индивидуальное или централизованное отопление -- что оптимально?

**Индивидуальное** -- для жилых домов. Причины: нет затрат на теплотрассу (15-40 млн руб), нет потерь при транспортировке (10-15%), нет единой точки отказа, каждый хозяин управляет своим отоплением.

**Централизованное** (от ТЭЦ) -- для кластера общественных зданий + теплиц. Здесь тепло -- побочный продукт выработки электричества, что радикально повышает экономику.

### 4. Покроют ли отходы лесопилки потребности в топливе?

**Частично** -- на 16-40%. Лесопилка при переработке 5 000-10 000 м3 кругляка даёт 800-2 000 тонн отходов. Общая потребность -- 5 000 тонн. Остальное -- дополнительная заготовка дров и низкосортной древесины. Это нормально: лесопилка не может быть единственным источником, но является важным компонентом топливного баланса.

### 5. Минимальная электрическая инфраструктура?

**Минимум для 300 домов:**
- 1 газификационная ТЭЦ 200 кВт_эл (основной источник)
- 1 дизельный генератор 100-150 кВт (резерв)
- Внутренняя сеть 0.4 кВ (~5-8 км кабеля)
- Стоимость минимума: ~20-30 млн руб

**Рекомендуемый вариант:**
- 2 x ТЭЦ 200 кВт_эл (400 кВт, с резервированием)
- Подключение к сети (если ЛЭП < 3 км)
- Стоимость: ~35-60 млн руб
